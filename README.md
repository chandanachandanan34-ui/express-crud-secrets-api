# 🚀 Node.js CRUD API Project

A full-stack CRUD application built using **Node.js, Express, and Axios** by integrating with an external API. This project demonstrates how to perform all RESTful operations and handle authentication using Bearer Token.

---

## 📌 Features

- 🔍 Fetch data using GET request  
- ➕ Create new data using POST  
- 🔄 Update entire data using PUT  
- ✏️ Partially update data using PATCH  
- ❌ Delete data using DELETE  
- 🔐 Secure API access using Bearer Token  
- 📄 Dynamic rendering using EJS  

---

## 🛠️ Tech Stack

- Node.js  
- Express.js  
- Axios  
- EJS  
- Body-parser

## ⚙️ Installation & Setup

1. Clone the repository  

git clone https://github.com/your-username/nodejs-crud-api-axios.git


2. Navigate to the project folder  

cd nodejs-crud-api-axios


3. Install dependencies  

npm install


4. Run the server  

node app.js


5. Open in browser  

http://localhost:3000


---

## 🔑 API Integration

This project uses an external Secrets API and performs authenticated requests using a Bearer Token.

---

## 📸 Usage

- Enter ID to fetch or delete data  
- Provide secret and score to create/update  
- Use PATCH to update specific fields only
